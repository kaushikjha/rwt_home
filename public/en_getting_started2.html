Getting started with Rwt

 

Now that our main menu is created, lets take care of our first window. Create a controller, let's say the test_controller.rb, with the following code:

  

class TestController < ApplicationController
  def first
    rwt_render
  end
end
and a file called 'first.rb' in the app/views/test directory with the code of a simple window: 

window('My first rwt window',300,200) do
  button('It seems to be easy') do |b|
    b.on('click') {show_view('/test/first')}
  end
end
This ends our first test. Try it with the 'Tests/First window' menu. You can have access to many other examples in the 'Rwt/Examples' menu.

The source code of Rwt is kept in http://github.com/smbrant/rwt.
