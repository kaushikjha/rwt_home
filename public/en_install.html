How to install this application (rwt_home)


  • $ git clone git://github.com/smbrant/rwt_home.git
  • $ cd rwt_home
  • $ mkdir log
  • $ mkdir vendor/plugins -p
  • $ cd vendor/plugins
  • $ git clone git://github.com/smbrant/rwt.git
  • $ cd ../..
  • $ rake rwt:install_extjs
  • $ ruby script/server
  • Browse to http://localhost:3000

Pre-requisites

  • Rails 2.2.2 installed
