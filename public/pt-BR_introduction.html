Este é o painel de visualização de código. Quando ativado ele mostra o código fonte que participa de cada exemplo.

O painel tenta não carregar o mesmo arquivo mais de uma vez. Você pode limpar as abas e desativar o painel se quiser.

Geralmente os exemplos são apresentados em duas versões. Uma é codificada em javascript usando a biblioteca ExtJs diretamente. A outra é codificada em ruby, usando o Rwt.

Você pode ver o primeiro exemplo clicando no link: uma janela simples.

Use a barra de tarefas para ter acesso aos outros exemplos.
