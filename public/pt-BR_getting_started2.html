Primeiros passos

 

Agora que o nosso menu principal está criado, vamos cuidar de nossa primeira janela. Crie um controlador, digamos o test_controller.rb, com o seguinte código:

  

class TestController < ApplicationController
  def first
    rwt_render
  end
end
e um arquivo chamado 'first.rb' no diretório the app/views/test com o código de uma janela simples: 

window('Minha primeira janela',300,200) do
  button('Parece ser fácil') do |b|
    b.on('click') {show_view('/test/first')}
  end
end
Assim terminamos o nosso primeiro teste. Experimente o menu 'Testes/Primeira janela'. Você pode ter acesso a muitos outros exemplos através do menu 'Rwt/Exemplos'.

O código fonte do Rwt é mantido em http://github.com/smbrant/rwt.
