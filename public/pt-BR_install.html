Como instalar esta aplicação (rwt_home)


  • $ git clone git://github.com/smbrant/rwt_home.git
  • $ cd rwt_home
  • $ mkdir log
  • $ mkdir vendor/plugins -p
  • $ cd vendor/plugins
  • $ git clone git://github.com/smbrant/rwt.git
  • $ cd ../..
  • $ rake rwt:install_extjs
  • #editar config/environment.rb se estiver utilizando MySQL (o default é Sqllite)
  • $ ruby db:create
  • $ ruby db:migrate
  • $ ruby script/server
  • Navegue em http://localhost:3000

Pré-requisitos

  • Rails 2.3.2 instalado
  • SqlLite (or MySQL)
