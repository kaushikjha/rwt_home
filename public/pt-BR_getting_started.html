Primeiros passos

O Rwt cria um ambiente onde é muito fácil desenvolver uma interface visual baseada na biblioteca javascript ExtJs e no Rails.

Para começar, você deve instalar o plugin Rwt em uma aplicação Rails:

$ cd my_app
$ ruby script/install http://rwt.accesstecnologia.com.br/plugin/get/rwt

Agora, baixe a biblioteca ExtJs de http://extjs.com e coloque-a em um subdiretório com o nome 'ext' no diretório public de sua aplicação (my_app/public/ext).

 

Nós devemos ter uma ação que crie uma aplicação Rwt. Você pode criar, por exemplo, um controlador chamado 'desktop' com uma ação 'index' com o seguinte código:

 

class DesktopController < ApplicationController
  def index
    respond_to do |format|
      format.html
      format.js {rwt_render}
    end
  end
end
