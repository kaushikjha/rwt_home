Primeiros passos

 

Crie também uma vista composta pelos arquivos 'index.html.erb' e 'index.rb' no diretório app/view/desktop com o seguinte código:

  

<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Transitional//EN">
<html>
  <head>
    <title>Rwt - Rails Web Toolkit</title>
    <meta http-equiv="Content-Type" content="text/html; charset=UTF-8">
    <%= rwt_javascript_libraries %>
    <script src="/desktop/index?format=js" type="text/javascript"></script>
  </head>
  <%= rwt_body %>
  </body>
</html>
e

rwt_app do
  toolbar do
    menu('Rwt') do
      menu_item('O que é isto?',call_view('/desktop/what'))
      menu_item('Primeiros passos',call_view('/desktop/started'))
      menu_item('Explorar',call_view('/desktop/explore'))
    end
    menu('Testes') do
      menu_item('Primeira janela',call_view('/test/first'))
    end
  end
end
