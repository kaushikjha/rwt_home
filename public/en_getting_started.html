Getting started with Rwt

Rwt creates an environment where it is very easy to develop a user interface based on the ExtJs javascript library and Rails.

To start with, you should install the rwt plugin in a rails application:

$ cd my_app
$ ruby script/plugin install http://rwt.accesstecnologia.com.br/plugin/get/rwt
$ rake rwt:install_extjs

Note that this rake command installs the minimum necessary files from version 2.2 of ExtJs. If you prefer you can download the extjs library from http://extjs.com and put it in a subdirectory named 'ext' in the public directory of your application (my_app/public/ext).

 

We should have an action that creates one rwt application. You could create, for example, a controller called 'desktop' with an 'index' action with the following code:

 

class DesktopController < ApplicationController
  def index
    respond_to do |format|
      format.html
      format.js {rwt_render}
    end
  end
end
